Introduction

This seems like the best way to learn CSS: create a single HTML page that contains all the different elements and design patterns that you'll likely use in a web design project. I've put this on Github too and welcome any feedback/criticism of these fumblings!

I'll link to tools, tutorials and relevant articles as I fumble along. The smily-face logo utilises the 'fontelico' font via weloveiconfonts.com and I used Adobe's colour wheel tool to pick the colour scheme.

Boxes in a row

This is intended to be a simple structure for creating rows of boxes. These are divs are set to display: inline-block; with a media query to switch to block representation for smaller screens. The width is set as a percentage with margin also as a percentage.

Block 1 has just one sentence in it.
Block 2 has more than one sentence. In fact it has multiple sentences because we want this to be the tallest block, height-wise. Hopefully this will do the trick.
Block 3 has two sentences. It should be taller than block 1 but not block 3.

The above example was a row of three boxes which I gave the class name .box-33. The '33' represents the approximate width of the box depending on how many you want in a row. More classes have been added to allow for different layouts: .box-25, .box-75, .box-66 and so on.

Each row of boxes (annoyingly) has to be wrapped in a div that has the .row { font-size: 0; } class; one of the solutions from CSS-tricks for how to fight unwanted gaps between inline-block elements.

There's a great tutorial at learnlayout.com which I found very useful as a newcomer.

A rant about boxes

I am perplexed by how difficult it is to put text in boxes and place those boxes exactly how you want them. I know there are frameworks and grid patterns and so on and so forth. But it is remarkably complicated considering the task at hand. And I'm not trying to be dismissive of CSS; it's just that if I spend half an hour to place these three divs neatly in a line and turn to my wife to share this triumph, you can't blame her for looking at me like "wtf..?"
